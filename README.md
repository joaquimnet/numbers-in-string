> Extract numbers from a string.

## Install

```sh
npm install numbers-in-string
```

## Usage

```javascript
numbers('I have 25 sandwiches and 8 sodas. It cost me $44.30.'); // -> [25, 8, 44.3]
```

## Run tests

```sh
npm test
```
